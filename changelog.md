# Changelog

Version history for LABsystem.

## [1.0.0] - 2025-11-19

### Initial Release

First public release of LABsystem - Professional Camera & Render Ecosystem for Blender.

**Core Features:**
- Modular architecture (CameraLAB, LightLAB, WorldLAB, FrameLAB, RenderLAB)
- Three access methods (Sidebar, LABwindow, LABmenu)
- Dynamic module enable/disable
- Customizable keyboard shortcuts

**CameraLAB:**
- Camera creation from viewport
- Camera switching and cycling
- Per-camera resolution presets
- Camera management tools

**RenderLAB:**
- Viewport-based rendering
- Automatic settings backup/restore
- Save/load render configurations
- Safe rendering with error recovery

**LightLAB:**
- Centralized light management
- Batch energy adjustments
- Quick light selection
- Light preset system

**WorldLAB:**
- Per-camera HDRI/World assignment
- Auto-sync when switching cameras
- Per-camera Strength control (0-10)
- Per-camera Rotation control (0-360Â°)
- HDRI import directly in addon
- World shader management (create, duplicate, rename, delete)
- Export world configurations
- Real-time viewport updates

**FrameLAB:**
- Frame range presets
- Quick timeline navigation
- Timeline marker management
- Playback control

**System Requirements:**
- Blender 3.0.0 or newer
- Windows, macOS, or Linux
