# Installation

## Install the Addon

1. Download LABsystem zip file
2. In Blender, go to **Edit > Preferences > Add-ons**
3. Click **Install** and select the zip file
4. Enable **LABsystem** in the addon list
5. Done!

## First Launch

Press **N** in the 3D View to open the sidebar. You'll see the **LABsystem** tab with all modules.

## Enable/Disable Modules

Go to **Edit > Preferences > Add-ons > LABsystem** to:

- Enable or disable individual modules (CameraLAB, RenderLAB, LightLAB, FrameLAB)
- Customize keyboard shortcuts
- Configure LABwindow and LABmenu shortcuts

All modules are enabled by default.

## Quick Access Setup

**LABwindow** (Default: F1)
- Floating window with all tools
- Can be customized in preferences

**LABmenu** (Default: Ctrl+Alt+Shift+Numpad 0)
- Radial marking menu
- Can be customized in preferences

That's it! You're ready to use LABsystem.
