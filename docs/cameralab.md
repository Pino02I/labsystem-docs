# CameraLAB

Professional camera management with resolution presets.

---

## Create Camera

### New from View
Creates a camera matching your current viewport position.

**How to use:**
1. Position viewport where you want
2. Click "New from View"
3. Camera created and set as active

**Status display:**
Shows current view type (Perspective View / Camera view with name)

## Resolution / Camera

Set resolution for the active camera.

**Shows:** Camera - ACTIVE

### Width / Height
Direct resolution input in pixels.

### Quick Presets (4 buttons)

- **DCI** - DCI 4K format
- **4K** - 4K UHD
- **9:16** - Vertical (mobile)
- **1:1** - Square (social media)

### More Presets
Click to expand additional resolution options.

**When expanded, shows categories:**
- Standard (HD, QHD, 768p)
- Cinema (Scope, Flat, 2K)
- Social (IG, 720p)
- Gaming (UW, SUW, UW FHD, Premium UW)
- Pro (ARRI, RED 8K, 2.4:1, Presentation)

## Viewport & Depth of Field

Control viewport behavior and DOF.

### Viewport & Lens

**Four camera type buttons:**
- **Perspective** - Standard perspective view
- **Orthographic** - Orthographic projection
- **Panoramic** - Panoramic camera
- **Custom** - Custom settings

**Viewport Lens:** 50 mm (adjustable)

### Lock Camera to View
When enabled, camera follows viewport navigation.

### Walk Navigation / Fly Navigation
Choose navigation mode when camera locked.

### Depth of Field

**Enable DOF**
Toggles depth of field on/off.

**Reset**
Resets DOF settings.

**Passepartout Alpha:** 0.500 (slider)
Controls viewport darkening outside camera view.

## Quick Reference

**Create workflow:**
1. Position viewport
2. Click "New from View"
3. Camera created at that view

**Resolution workflow:**
1. Select camera
2. Choose quick preset (DCI, 4K, 9:16, 1:1)
3. Or use "More Presets" for specific formats

**DOF workflow:**
1. Enable DOF
2. Adjust Passepartout Alpha for viewport preview
3. Set focus distance in camera settings

**Quick positioning:**
1. Select object
2. Use "Frame Selected"
3. Camera frames the object

**Lock to view:**
1. Enable "Lock Camera to View"
2. Navigate viewport
3. Camera follows movement
4. Use Walk or Fly navigation mode

**Pro tips:**
- Use quick presets for common formats
- Lock camera when positioned correctly
- Show camera names when working with multiple cameras
- Use Frame Selected for quick object framing
- Passepartout helps visualize final frame
