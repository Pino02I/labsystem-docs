# LABsystem v1.0.0

Professional Camera & Render Ecosystem for Blender

LABsystem is a comprehensive, modular addon for Blender that streamlines camera management, rendering workflows, lighting setups, and frame range operations. Built with a professional production pipeline in mind, LABsystem offers an integrated ecosystem of tools accessible through an intuitive interface.

## Features

LABsystem consists of four modular components that work together seamlessly:

### CameraLAB
Advanced camera management system with professional features:
- **Camera Manager**: Organize and switch between multiple cameras effortlessly
- **Resolution Memory**: Store and recall custom resolution presets per camera
- **Advanced Operators**: Quick camera creation, duplication, and manipulation
- **Scene Management**: Efficient camera switching and viewport synchronization

### RenderLAB
Professional rendering toolkit with intelligent automation:
- **Auto-Restore System**: Automatically saves and restores render settings
- **Hotkey System**: Customizable keyboard shortcuts for rendering operations
- **Batch Operations**: Streamlined multi-camera and multi-scene rendering
- **Settings Management**: Store and load render configurations

### LightLAB
Comprehensive lighting management tools:
- **Light Manager**: Control all scene lights from a centralized interface
- **Quick Setup**: Rapid lighting presets and configurations
- **Batch Operations**: Modify multiple lights simultaneously
- **Energy Management**: Fine-tune light intensities across your scene

### FrameLAB
Frame range management and animation helpers:
- **Range Presets**: Save and load custom frame ranges
- **Quick Navigation**: Jump to specific frames instantly
- **Timeline Control**: Efficient animation workflow management
- **Scene Coordination**: Synchronize frame ranges across multiple scenes



## Access Points

LABsystem provides multiple ways to access its tools:

1. **Sidebar Panel**: `3D View > Sidebar (N) > LABsystem Tab`
2. **LABwindow Popup**: Press `F1` (customizable) for quick access
3. **LABmenu Marking Menu**: Press `Ctrl+Alt+Shift+Numpad 0` (customizable) for radial menu
4. **Module Controls**: Enable/disable individual modules in addon preferences



## Module System

LABsystem features a modular design that allows you to:

- **Enable/Disable Modules**: Activate only the tools you need
- **Customize Workflow**: Tailor LABsystem to your specific production requirements
- **Reduce Overhead**: Minimize memory and UI clutter by disabling unused modules
- **Modular Updates**: Each module can be updated independently

To manage modules:
1. Go to `Edit > Preferences > Add-ons`
2. Find "LABsystem" in the list
3. Expand the addon preferences
4. Toggle individual modules on/off
5. Modules activate/deactivate immediately without restart

## Documentation

### Quick Start
1. Install LABsystem using the Installation Guide
2. Press `N` in 3D View to open the sidebar
3. Navigate to the LABsystem tab
4. Enable desired modules in addon preferences
5. Start using the tools!

### Keyboard Shortcuts
- `F1`: Open LABwindow popup (default, customizable)
- `Ctrl+Alt+Shift+Numpad 0`: Open LABmenu marking menu (default, customizable)

### Preferences
Access addon preferences through:
`Edit > Preferences > Add-ons > LABsystem`

Here you can:
- Enable/disable individual modules
- Customize keyboard shortcuts
- Configure auto-save and auto-restore behaviors
- Set default values for various operations



## Troubleshooting

### Module Not Appearing
- Check if the module is enabled in addon preferences
- Restart Blender after enabling/disabling modules
- Check the console for error messages

### Keyboard Shortcuts Not Working
- Ensure shortcuts don't conflict with other addons
- Customize shortcuts in addon preferences if needed
- Check if the correct context is active (3D View)

### Console Messages
LABsystem provides detailed console output:
- Success messages in green
- ⚠️ Warning messages in yellow
- ❌ Error messages with traceback information

To view console: `Window > Toggle System Console` (Windows) or launch Blender from terminal (Linux/macOS)
