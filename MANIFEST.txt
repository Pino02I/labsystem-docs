## File Usage

### For Addon Distribution (Blender Market, Gumroad, etc.)
Include these files with your addon ZIP:
- README.md
- INSTALLATION.md
- USER_GUIDE.md
- DOCUMENTATION.md
- CHANGELOG.md
- LICENSE

### For Online Documentation (Read the Docs)
Use these files:
- .readthedocs.yaml
- docs/ folder (entire directory)
Follow READTHEDOCS_SETUP.md for configuration

### For GitHub Repository
Include all files:
- All documentation files
- .readthedocs.yaml
- docs/ folder
- README.md as repository main page
